## Overview

LinkSpot supports seamless integrations with popular third-party services, webhooks for automation, and a robust API for custom extensions. You can embed external content directly into your docs and sync changes with version control systems. These features help you build a connected documentation ecosystem without leaving your workflow.

<Callout kind="info">
Review the integration permissions in your LinkSpot dashboard at `https://dashboard.example.com/integrations` before connecting services.
</Callout>

## Set Up Webhooks for Automation

Webhooks enable real-time notifications when events occur in LinkSpot, such as page updates or new comments.

<Steps>
  <Step title="Create Webhook" icon="plus">
    Navigate to `https://dashboard.example.com/webhooks` and click **New Webhook**.
  </Step>
  <Step title="Configure Endpoint" icon="link">
    Enter your endpoint URL, like `https://your-webhook-url.com/webhook`.
  </Step>
  <Step title="Select Events" icon="settings">
    Choose events such as `page.updated` or `comment.created`.
  </Step>
  <Step title="Test and Save" icon="check-circle">
    Send a test payload and save the webhook.
  </Step>
</Steps>

Here is a sample webhook payload:

```json
{
  "event": "page.updated",
  "data": {
    "pageId": "doc-123abc",
    "title": "API Reference",
    "url": "https://docs.example.com/api"
  },
  "timestamp": "2024-10-15T12:00:00Z"
}
```

## Custom Extensions via API

Build custom integrations using LinkSpot's REST API at `https://api.example.com/v1`.

<ParamField path="projectId" param-type="string" required="true">
  Your LinkSpot project identifier.
</ParamField>

<ParamField header="Authorization" param-type="string" required="true">
  Bearer token: `Bearer YOUR_API_KEY`.
</ParamField>

<CodeGroup tabs="JavaScript,cURL">
  ```javascript
  const response = await fetch('https://api.example.com/v1/pages', {
    headers: {
      'Authorization': 'Bearer YOUR_API_KEY',
      'Content-Type': 'application/json'
    }
  });
  const pages = await response.json();
  console.log(pages);
  ```
  ```bash
  curl -X GET https://api.example.com/v1/pages \
    -H "Authorization: Bearer YOUR_API_KEY" \
    -H "Content-Type: application/json"
  ```
</CodeGroup>

## Embed External Content

Embed videos, forms, or dashboards directly into your LinkSpot pages.

<Tabs>
  <Tab title="YouTube Video" icon="video">
    Use an iframe embed code:

````html
<iframe
  width="560"
  height="315"
  src="https://www.youtube.com/embed/dQw4w9WgXcQ"
  title="Sample Video"
  frameborder="0"
  allowfullscreen>
</iframe>
````

  </Tab>
  <Tab title="Google Form" icon="file-text">
    Paste the embed snippet:

````html
<iframe
  src="https://docs.google.com/forms/d/e/YOUR_FORM_ID/viewform?embedded=true"
  width="640"
  height="800">
  Loading…
</iframe>
````

  </Tab>
</Tabs>

## Sync with Version Control

Keep your docs in sync with Git providers.

<Expandable title="GitHub Sync" default-open="true">
  Connect your repo at `https://dashboard.example.com/integrations/github`.

  ```yaml
  # .linkspot.yml in your repo root
  sync:
    branch: main
    path: docs/
  ```
</Expandable>

<Expandable title="GitLab Sync">
  Use webhooks or API polling for GitLab repos. Configure in project settings.
</Expandable>
