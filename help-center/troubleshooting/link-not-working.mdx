---
title: Link Not Working
description: Fix links that don't open as expected.
---

## Symptoms

- Clicking a link shows an error
- Link opens a blank page

## Solution

<Steps>
  <Step title="Check URL format">
    The link must start with `http://` or `https://`.
  </Step>
  <Step title="Test link">
    Paste the URL in your browser to verify it loads.
  </Step>
  <Step title="Edit link in LinkSpot">
    Use the edit option to fix any typos.
  </Step>
</Steps>
