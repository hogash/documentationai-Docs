## Recent Releases

Stay informed about all changes to LinkSpot. Review the latest releases below, organized by version. Each release includes new features, improvements, bug fixes, and any breaking changes.

<Update label="2024-10-15" description="v1.2.0" tags={["feature", "bugfix", "improvement"]}>

## New Features

- Added real-time collaboration for document editing with live cursors and comments.
- Introduced custom themes and branding options for documentation spaces.
- New API endpoints for programmatic document management.

## Improvements

- Enhanced search performance with fuzzy matching and full-text indexing.
- Improved mobile responsiveness across all documentation pages.

## Bug Fixes

- Fixed issue where embedded images failed to load in certain browsers.
- Resolved synchronization delays in multi-user editing sessions.

</Update>

<Update label="2024-09-20" description="v1.1.0" tags={["feature", "breaking"]}>

## New Features

- Integrated version history with diff views and restore functionality.
- Added webhook support for external notifications on document changes.

## Breaking Changes

- Updated API authentication to require `{API_KEY}` in headers instead of query params. Migrate your integrations using the new format.

## Bug Fixes

- Corrected export to PDF rendering issues with complex tables.
- Fixed navigation menu collapse on smaller viewports.

</Update>

<Update label="2024-08-10" description="v1.0.0" tags={["feature"]}>

## Initial Release

- Core documentation workspace with page creation, organization, and sharing.
- Rich text editor with markdown support and embeds.
- User authentication and role-based access control.
- Basic analytics dashboard for page views.

</Update>

## Upgrade Instructions

Follow these steps to upgrade LinkSpot to the latest version.

<Steps>
  <Step title="Backup Your Data" icon="database">

    Create a full export of your documentation space.

````bash
linkspot export --space-id YOUR_SPACE_ID --output backup.json
````

  </Step>
  <Step title="Update CLI" icon="download">

    <CodeGroup tabs="npm,yarn">
````bash
npm install -g @linkspot/cli@latest
````
````bash
yarn global add @linkspot/cli@latest
````
    </CodeGroup>

  </Step>
  <Step title="Run Migration" icon="settings">

    Apply any database migrations if applicable.

````bash
linkspot migrate --api-key YOUR_API_KEY
````

  </Step>
  <Step title="Verify" icon="check-circle">

    Test key features like editing and sharing.

  </Step>
</Steps>

<Callout kind="tip">
  For self-hosted installations, review the [migration guide](https://docs.example.com/self-hosted/migration) for version-specific steps.
</Callout>
