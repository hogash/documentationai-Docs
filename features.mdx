## Overview

LinkSpot provides four types of interactive elements you can place on product gallery images. Each element type serves a different purpose, from cross-selling to feature callouts.

## Pins

Pins are clickable hotspots placed on product images that link to other products. They are ideal for lifestyle photos, room scenes, or any image where multiple products are visible.

### Pin Styles

LinkSpot offers 12 pin styles to match your store's aesthetic:

| Category | Styles |
|----------|--------|
| Classic | 5 variants |
| Geometric | Disk, Plus (2 variants), Dashed |
| Decorative | Neon, Thumb, X |
| Custom | Upload your own SVG icon |

<Callout kind="info">
All 12 pin styles are available on Growth and Scale plans. Free and Starter plans include a subset of styles.
</Callout>

### Pin Interaction

Configure how customers interact with pins:

| Setting | Options | Description |
|---------|---------|-------------|
| Interaction event | Click, Hover | Choose the mouse event that opens the product card |
| Hover animation | On/Off | Animate the pin when hovered |
| Constant animation | On/Off | Keep pins in constant motion (e.g., pulsating) to attract attention |

### Pin Limits

Each plan specifies how many products can have pins and how many pins can be placed per image. See [Plans](#plans) below.

## Product Sets

Product Sets let you display curated groups of products on a product image. They are the primary tool for cross-selling and upselling.

### Layouts

<Tabs>
  <Tab title="Button & Popup" icon="mouse-pointer">
    A button is placed on the product image. When clicked, it opens a popup displaying the selected products.

    **Configuration options:**
    - Button position (Top Left, Top Right, Bottom Left, Bottom Right)
    - Button icon and color scheme
    - Popup color scheme
    - Cart behavior: "Add selected with one button" (best for outfits/bundles) or "Add each product separately" (best for recommendations)
  </Tab>

  <Tab title="Card Carousel" icon="columns">
    A horizontal carousel of product cards displayed on or near the product image.

    **Configuration options:**
    - Card size (Small, Medium, Large)
    - Card width
    - Content vertical alignment (Spaced, Centered)
    - Color scheme
  </Tab>

  <Tab title="Links List" icon="list">
    A list of product links displayed on the product image.

    **Configuration options:**
    - Alignment (Left, Right)
    - Item size (Small, Medium)
    - Color scheme
  </Tab>
</Tabs>

### Targeting

Product Sets can be targeted to specific products in several ways:

- **All products** -- the set appears on every product page
- **Specific collections** -- the set appears on products in selected collections
- **Specific products** -- the set appears only on hand-picked products
- **Product metafield** (Scale plan) -- assign a product set via the `LinkSpot Product Set` metafield on individual products

### Product Selection

Choose which products appear inside a product set:

- **Manual selection** -- hand-pick products from your catalog
- **Automatic** (Starter plan and above) -- use Shopify's Search & Discovery recommendations to automatically populate complementary or related products

### Image Placement

Product Sets can be targeted to a specific image position in the product gallery (e.g., the 3rd image). If the product doesn't have that many images, you can configure the set to either hide or fall back to the last available image.

## Annotations

Annotations are text overlays placed on product images to highlight features, materials, dimensions, or other details.

### Configuration

| Setting | Options | Description |
|---------|---------|-------------|
| Intro transition | Fade, Slide, None | Animation when the annotation first appears |
| Color scheme | Customizable | Background and text colors |
| Position | Click to place | Position by clicking on the product image |

Annotations are positioned freely on the image -- click where you want the annotation to appear and enter your text.

## Captions

Captions are text overlays that display in a corner of the product image.

### Configuration

| Setting | Options | Description |
|---------|---------|-------------|
| Size | Small, Medium, Large, Larger | Text and padding size |
| Position | Top Left, Top Right, Bottom Left, Bottom Right | Corner placement |
| Color scheme | Customizable | Background and text colors |

## Plans

LinkSpot offers four plans. Feature limits scale with each tier.

| Feature | Free | Starter | Growth | Scale |
|---------|------|---------|--------|-------|
| Product Sets | Limited | More | More | Most |
| Automatic product sets | -- | Yes | Yes | Yes |
| Advanced targeting | -- | -- | Yes | Yes |
| Scalable targeting & metafield assignment | -- | -- | -- | Yes |
| Products with Pins | Limited | More | More | Most |
| Products with Annotations | Limited | More | More | Most |
| Products with Captions | Limited | More | More | Most |
| All 12 pin styles | -- | -- | Yes | Yes |
| Multi-language support | -- | -- | Yes | Yes |
| Theme compatibility setup | Yes | Yes | Yes | Yes |
| Email support | Yes | Yes | Yes | Yes |

<Callout kind="info">
The Free plan includes LinkSpot branding on elements. Paid plans remove the branding.
</Callout>

## General Settings

These settings apply across all element types:

| Setting | Options | Description |
|---------|---------|-------------|
| Product card click behavior | Open in new tab, Open product popup | What happens when a customer clicks a product card |
| Show product vendor | On/Off | Display the vendor name in product presentations |
| Show currency code | On/Off | Show currency code (e.g., USD) in prices |
| Corner style | Square, Soft, Rounded | Controls corner radius on buttons, cards, and popups |
| Color scheme | Light, Dark, Auto | Product quick-view popup color scheme |
