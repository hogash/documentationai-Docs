## Overview

LinkSpot turns product images into interactive sales assets, using shoppable elements for cross-selling and visual overlays to explain product features directly on the image.

Add clickable hotspots, curated product groups, text callouts, and captions to any product gallery image, all without editing your theme code. LinkSpot works through a theme app embed that integrates seamlessly with your Shopify storefront.

## Quick Start

Get up and running in minutes with these steps.

<Steps>
  <Step title="Install LinkSpot" icon="download" title-type="p">
    Install LinkSpot from the Shopify App Store. Approve the required permissions when prompted.
  </Step>

  <Step title="Enable the App Embed Block" icon="toggle-right" title-type="p">
    Open your theme editor and enable the LinkSpot app embed block, then save. This allows LinkSpot to load on your storefront.
  </Step>

  <Step title="Run Compatibility Test" icon="check-circle" title-type="p">
    From the LinkSpot dashboard, run the compatibility test to verify your theme works with LinkSpot. The test detects your product image gallery automatically.
  </Step>

  <Step title="Preview in the storefront" icon="plus" title-type="p">
    Preview a product page in your storefront to ensure LinkSpot correctly located your product image gallery and is able to place the elements inside. You should see some sample elements floating on the images. If not, please contact support so we can investigate why this happens.
  </Step>
</Steps>

<Tabs>

## How It Works

LinkSpot uses a Shopify theme app embed to render interactive elements on your product pages.

```mermaid
graph TD
    ShopifyStore[Shopify Store] --> ThemeExtension[LinkSpot App Embed block]
    ThemeExtension --> ProductPages[Product Pages]
    ProductPages --> Pins[Pins]
    ProductPages --> ProductSets[Product Sets]
    ProductPages --> Annotations[Annotations]
    ProductPages --> Captions[Captions]
```

<Expandable title="Technical Details" default-open="false">
  LinkSpot stores element data in Shopify metafields. The app embed block reads these metafields and renders the interactive elements on the product gallery images. No theme code modifications are required.
</Expandable>
